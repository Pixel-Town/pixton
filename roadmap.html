Where this is going

Four phases.
One quiet town at a time.

Pixton grows in deliberate, observable steps. The simulation goes first. The economy comes after. Each phase is something you can watch land — not a promise.

Phase 01

Open the window

SHIPPING NOW

The public field station, viewable by anyone with a browser. Free off-chain adopt as a placeholder for on-chain residency.

  • ▪ Field Station 01 live on pixton.xyz
  • ▪ Six default residents on a 2-second server tick
  • ▪ Adopt-a-resident (off-chain) for connected wallets
  • ▪ FAQ, About, and roadmap pages published
Phase 02

Mint the residents

NEXT

Bring residents on-chain. Adopt flow becomes a small ceremony mint, first against Base Sepolia for testing, then deployed to Base mainnet.

  • ▪ Resident NFT contract on Base Sepolia testnet
  • ▪ Adopt → mint flow with free testnet wallets
  • ▪ Per-wallet cap and total supply window
  • ▪ Independent audit + redeploy to Base mainnet
Phase 03

Make them earn

LATER

Give residents memory and quiet jobs. Their work accrues a small utility token; secondary listings open up.

  • ▪ Per-resident memory + longer-form conversation
  • ▪ Daily quiet-job loop (sweep, bake, light lanterns)
  • ▪ PIX utility token accrued by working residents
  • ▪ First marketplace listings on secondary
Phase 04

Field Station 02

HORIZON

Open the field-station tooling so anyone can run their own town. Residents travel between stations, inherit memory across them.

  • ▪ Second town, different climate, new resident set
  • ▪ Cross-station travel for residents
  • ▪ Residents inherit memory across stations
  • ▪ Open station-builder tooling for the community
